## Pre-requisites

- [x] A VM or a physical machine with Ubuntu 22.04
- [x] Docker
- [x] Docker Compose

### Install Docker

```bash
curl -sSL https://get.docker.com | sh
```

### Install Docker Compose

```bash
apt update && apt install -y docker-compose
```
